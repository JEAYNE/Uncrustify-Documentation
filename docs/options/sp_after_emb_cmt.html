sp_after_emb_cmt

Properties

  Category: Spacing options
      Type: multiple (ignore add remove force)
   Default: force

Description

Add or remove space after an embedded comment.

Default: force

Examples

raw CPP code sp_after_emb_cmt=ignore
sp_after_type=ignore		 sp_after_emb_cmt=add
sp_after_type=ignore		 sp_after_emb_cmt=remove
sp_after_type=ignore		 sp_after_emb_cmt=force
sp_after_type=ignore
