sp_special_semi

Properties

  Category: Spacing options
      Type: multiple (ignore add remove force)
   Default: ignore

Description

Add or remove space before empty statement ';' on 'if', 'for' and 'while'.

Examples

raw CPP code sp_special_semi=ignore sp_special_semi=add sp_special_semi=remove sp_special_semi=force space
