align_asm_colon

Properties

  Category: Code alignment options (not left column spaces/tabs)
      Type: boolean [false, true]
   Default: false

Description

Whether to align text after 'asm volatile ()' colons.

Examples

raw CPP code align_asm_colon=false align_asm_colon=true space
align_asm_colon=true
