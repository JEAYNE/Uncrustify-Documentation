sp_enum_after_assign

Properties

  Category: Spacing options
      Type: multiple (ignore add remove force)
   Default: ignore

Description

Add or remove space after assignment '=' in enum.

Overrides sp_enum_assign.

Examples

raw CPP code sp_enum_after_assign=ignore sp_enum_after_assign=add sp_enum_after_assign=remove sp_enum_after_assign=force space
sp_enum_after_assign=force
enum{A,B,C=-2,D,E}; enum {A,B,C=-2,D,E}; enum {A,B,C= -2,D,E}; enum {A,B,C=-2,D,E}; enum {A,B,C= -2,D,E}; enum{A,B,C=-2,D,E};
