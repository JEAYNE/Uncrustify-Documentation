indent_case_brace

Properties

  Category: Indenting options
      Type: numeric [-16 .. 16]
   Default: 0

Description

Spaces to indent '{' from 'case'. By default, the brace will appear under
the 'c' in case. Usually set to 0 or indent_columns. Negative values are OK.
It might be wise to choose the same value for the option indent_switch_case.

Examples

raw CPP code indent_case_brace=0
void f(){switch(a){case 1:a++;break;case -1:a--;case 123:a--;break;default:a=0;}} void f(){ switch(a){case 1: a++; break; case -1: a--; case 123: a--; break; default: a=0;} }
