nl_multi_line_cond

Properties

  Category: Newline adding and removing options
      Type: boolean [false, true]
   Default: false

Description

Add a newline between ')' and '{' if the ')' is on a different line than the
if/for/etc.

Overrides nl_for_brace, nl_if_brace, nl_switch_brace, nl_while_switch and
nl_catch_brace.

raw CPP code nl_multi_line_cond=false nl_multi_line_cond=true nl
nl_multi_line_cond=true
