sp_pp_concat

Properties

  Category: Spacing options
      Type: multiple (ignore add remove force)
   Default: add

Description

Add or remove space around preprocessor '##' concatenation operator.

Default: add

a##b

Examples

raw CPP code sp_pp_concat=ignore sp_pp_concat=add sp_pp_concat=remove sp_pp_concat=force space
sp_pp_concat
