nl_after_namespace

Properties

  Category: Blank line options
      Type: numeric [0 .. 16]
   Default: 0

Description

The number of newlines after '}' of a namespace.

Examples

raw CPP code nl_after_namespace=0 nl_after_namespace=5 nl_after_namespace=10 nl_after_namespace=16 nl
