nl_squeeze_ifdef_top_level

Properties

  Category: Newline adding and removing options
      Type: boolean [false, true]
   Default: false

Description

Makes the nl_squeeze_ifdef option affect the top-level #ifdefs as well.

Examples

raw CPP code nl_squeeze_ifdef_top_level=false nl_squeeze_ifdef_top_level=true nl
