sp_before_comma

Properties

  Category: Spacing options
      Type: multiple (ignore add remove force)
   Default: remove

Description

Add or remove space before ',', i.e. 'a,b' vs. 'a ,b'.

Default: remove

int a,b;
f(a,b);

Examples
