nl_after_brace_open

Properties

  Category: Newline adding and removing options
      Type: boolean [false, true]
   Default: false

Description

Whether to add a newline after '{'.

Examples

raw CPP code nl_after_brace_open=false nl_after_brace_open=true nl
nl_after_brace_open=true
