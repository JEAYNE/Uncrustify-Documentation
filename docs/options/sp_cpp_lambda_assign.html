sp_cpp_lambda_assign

Properties

  Category: Spacing options
      Type: multiple (ignore add remove force)
   Default: ignore

Description

Add or remove space around '=' in C++11 lambda capture specifications.

Overrides sp_assign.

[=](function_parameters) -> return_type { body }
