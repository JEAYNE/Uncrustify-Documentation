sp_addr

Properties

  Category: Spacing options
      Type: multiple (ignore add remove force)
   Default: remove

Description

Add or remove space after the '&' (address-of) unary operator. This does not
affect the spacing after a '&' that is part of a type.

Default: remove

ptr=&var
