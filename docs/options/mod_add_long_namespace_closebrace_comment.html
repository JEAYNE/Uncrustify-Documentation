mod_add_long_namespace_closebrace_comment

Properties

  Category: Code modifying options (non-whitespace)
      Type: numeric [0 .. 255]
   Default: 0

Description

If a namespace body exceeds the specified number of newlines and doesn't
have a comment after the close brace, a comment will be added.

Examples

raw CPP code mod_add_long_namespace_closebrace_comment=0
using namespace std::string_literals; namespace A{int i;}; namespace B{int i;int j;namespace C{namespace D{int k;}}} using namespace std::string_literals; namespace A {int i;}; namespace B {int i; int j; namespace C {namespace D {int k;}}}
 

Info

  need to be tuned with some values in [0 .. 255]
