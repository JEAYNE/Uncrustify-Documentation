nl_max

Properties

  Category: Blank line options
      Type: numeric [0 .. 16]
   Default: 0

Description

The maximum number of consecutive newlines (3 = 2 blank lines).

Examples

raw CPP code nl_max=0 nl_max=9 nl_max=6 nl_max=3 nl
