nl_enum_colon_type

Properties

  Category: Newline adding and removing options
      Type: multiple (ignore add remove force)
   Default: ignore

Description

Add or remove newline between 'enum class identifier :' and type.

Examples

raw CPP code nl_enum_colon_type=ignore nl_enum_colon_type=add nl_enum_colon_type=remove nl_enum_colon_type=force nl
nl_enum_colon_type
enum{A,B,C=-2,D,E}; enum {A,B,C=-2,D,E}; enum {A,B,C=-2,D,E}; enum {A,B,C=-2,D,E}; enum {A,B,C=-2,D,E}; enum {A,B,C=-2,D,E};
