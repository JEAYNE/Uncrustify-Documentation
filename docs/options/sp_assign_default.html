sp_assign_default

Properties

  Category: Spacing options
      Type: multiple (ignore add remove force)
   Default: ignore

Description

Add or remove space around assignment operator '=' in a prototype.

If set to ignore, use sp_assign.

void f(int a=3);

Examples

raw CPP code sp_assign_default=add sp_assign_default=remove sp_assign_default=force space
