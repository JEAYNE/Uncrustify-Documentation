nl_split_for_one_liner

Properties

  Category: Newline adding and removing options
      Type: boolean [false, true]
   Default: false

Description

Whether to split one-line simple unbraced for statements into two lines by
adding a newline, as in 'for (...) <here> stmt;'.

Examples

raw CPP code nl_split_for_one_liner=false nl_split_for_one_liner=true nl
