nl_max_after_func_body

Properties

  Category: Blank line options
      Type: numeric [0 .. 16]
   Default: 0

Description

The maximum number of newlines after '}' of a multi-line function body.

Only works when nl_after_func_body is 0.
Takes precedence over nl_min_after_func_body.

Examples

raw CPP code nl_max_after_func_body=0 nl_max_after_func_body=5 nl_max_after_func_body=10 nl_max_after_func_body=16 nl
