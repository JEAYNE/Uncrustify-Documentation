nl_before_struct

Properties

  Category: Blank line options
      Type: numeric [0 .. 16]
   Default: 0

Description

The number of newlines before a struct definition.

Examples

raw CPP code nl_before_struct=0 nl_before_struct=5 nl_before_struct=10 nl_before_struct=16 nl
