Before to start, please read this page about U.D.P.

The core of this project is:

  • The indexes of all Uncrustify options sorted by
    - name,
    - category,
    - type,
    - keyword,
    - exceptions to the default values.

  • The Comparison table of various configurations.
    K&R vs Linux kernel vs FreeBsd vs GNU vs ...

If it seems to you that Uncrustify is doing unexpected things,
this may be due to one of the options with a non standard default.

If you start with uncrustify I suggest that you
  • Take a look to the various indexes (see above)
  • Read the tuto how to write an uncrustify configuration from scratch
  • Read the explanations about parenthesis, gap-span-threshold and iarf options.
  • If you don't know which format to use, look at this list of recommendations ;-)
For curious there are these tools:
  • uncrustify online. Based on uncrustify 0.72 (2020/nov), but not maintained.
  • UniversalIndent GUI, but not maintained. The source is also available on github
  • Univerl-Ident-GUI-tqt is a fork of the UniversalIndent GUI made by Michele Calgaro, one of the maintainers of uncrustify.
